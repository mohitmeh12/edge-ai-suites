ROS2 KPI Monitoring
===================

The ROS2 KPI Monitoring toolkit provides real-time monitoring, analysis, and
visualization of Key Performance Indicators in ROS2 systems — including node
latencies, CPU/memory usage, message flow, and thread-level resource
distribution across local and remote machines.
